1. Install node.js from https://nodejs.org/en/download/
2. Install mysql server from https://dev.mysql.com/downloads/mysql/
3. Create a mysql user and grant all privileges
3. Install express (npm install express) and cors (npm install cors) and sql (npm install mysql) modules using npm 
4. In the properties.js file, make changes to the connection variable such that the host, port, user, and password match with the user in your system
5. Run 'node loader.js' in the terminal
6. Run 'node api.js' in the terminal
